# Introduction

A design system is a great way to keep everyone aligned. It consists of UX principles, style guides, and developer documentation needed to build consistent, on-brand and user-friendly products.

# Why use design system?

With no common design language to unite the product, the user experience starts to break down, as does the design process. Design critiques become unproductive when there’s a dearth of design conventions. To create alignment within teams, there must be a shared source of truth—a place to reference official patterns and styles. Design Systems Handbook - Invision
